# Sales Forecasting Project Summary

## Project Objective
The objective of this project is to analyze historical retail sales data and build forecasting models to predict future sales trends. The project also includes anomaly detection, customer segmentation, and an interactive dashboard for business insights.

## Techniques Used
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Time Series Analysis
- SARIMA Forecasting
- Facebook Prophet Forecasting
- XGBoost Regression
- Isolation Forest Anomaly Detection
- Z-Score Anomaly Detection
- K-Means Customer Segmentation
- Streamlit Dashboard

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Scikit-learn
- Statsmodels
- Prophet
- XGBoost
- Streamlit

## Key Outcome
The project demonstrates how machine learning and statistical forecasting techniques can be applied to historical retail sales data to generate forecasts, detect anomalies, and provide decision support through interactive visualizations.
