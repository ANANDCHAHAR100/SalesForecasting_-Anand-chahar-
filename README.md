# Sales Forecasting Project

## Overview
This project focuses on forecasting retail sales using historical Superstore sales data. Multiple forecasting techniques were implemented and compared to analyze future sales trends and business performance.

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SARIMA Forecasting
- Prophet Forecasting
- XGBoost Forecasting
- Z-Score Anomaly Detection
- Isolation Forest Anomaly Detection
- Customer Segmentation
- Streamlit Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Scikit-learn
- Statsmodels
- Prophet
- XGBoost
- Streamlit
